Installation Guide

Windows

Step 1: Download Python

Visit python.org and download the latest Python installer. Run the installer and check "Add Python to PATH".

Step 2: Open CMD

Press Win + R, type "cmd", and press Enter to open Command Prompt.

Step 3: Run the Script

Navigate to the script folder using cd path/to/tamil-mp3-downloader and run python main-classic.py.

macOS

Step 1: Install Homebrew

Open Terminal and run:
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Step 2: Install Python

Run brew install python3 to install Python via Homebrew.

Step 3: Run the Script

Navigate to the script directory with cd path/to/tamil-mp3-downloader and execute python3 main-classic.py.
