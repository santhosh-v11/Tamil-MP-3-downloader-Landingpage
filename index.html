Installation

Windows

Step 1: Download Python

Visit python.org and download the latest Python installer. Run the installer and check "Add Python to PATH".

Step 2: Open CMD

Press Win + R, type "cmd", and press Enter to open Command Prompt.

Step 3: Run the Script

Navigate to the script folder using cd path/to/tamil-mp3-downloader and run python main-classic.py.
